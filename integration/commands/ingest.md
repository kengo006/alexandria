# /ingest — enter Librarian mode

A slash-command entry point for the most frequent maintenance session. Install as `.claude/commands/ingest.md`.

```markdown
Enter Librarian mode. Read, in order:
0. {your governance layer, if any}
1. governance/role-division.md
2. roles/librarian.md
3. obsidian/note-schema.md, obsidian/vault-structure.md, obsidian/wikilinks-and-mocs.md
4. notes/vault-map.md
5. the errata queue, if one is pending

Then reply exactly: "📚 Librarian ready — inbox, errata, or maintenance?"
and wait for instructions. Remember: gates before notes; serial when content
depends on the source; the three-part completion report before "done".
```

The fixed ready-line is deliberate: it confirms the reading happened and telegraphs the three standard job types. Adapt the wording; keep the confirmation pattern.
